üåê Apa itu HTML?

HTML (Hypertext Markup Language) adalah bahasa markup yang digunakan untuk membangun halaman web dan menampilkan informasi di dalam browser. HTML saat ini menjadi standar yang diatur oleh World Wide Web Consortium (W3C).

üìÖ Sejarah HTML

  • HTML 2.0 disahkan pada 22 September 1995
  • HTML 3.2 diperkenalkan pada 14 Januari 1996
  • HTML 4.0 dirilis pada 18 Desember 1997
  • XHTML 1.0, gabungan HTML dan XML, muncul pada 24 Januari 2000

üìÑ Struktur HTML

<html>
    <head>
        <title>Judul Halaman</title>
    </head>
    <body>
        ...konten halaman...
    </body>
</html>

üìë Ciri-Ciri Dokumen HTML

    üìù Memiliki ekstensi file berupa .htm atau .html.
    üî§ Bersifat non case-sensitive, artinya tag dapat ditulis dalam huruf besar maupun kecil.
    üîó Terdiri atas tag pembuka dan penutup, walaupun beberapa tag bisa berdiri sendiri (seperti <br />).
    üìö Tag HTML bersifat berpasangan & bersarang, seperti tag <html> yang membungkus seluruh isi dokumen.

üîß Tag HTML Penting

  • Tag Heading: <h1> sampai <h6> untuk judul dan subjudul
  • Tag Paragraf: <p> untuk menulis paragraf
  • Tag Gambar: <img src="..." alt="..."> untuk menampilkan gambar
  • Tag Hyperlink: <a href="...">teks link</a> untuk tautan
  • Tag Komentar: <!-- ... --> untuk menyisipkan komentar
